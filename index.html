Basic Skills

Basic Skills akan mulai dipelajari di kelas 10, lalu apa aja si yang mencangkup basic skills ini? ada beberapa diantaranya yaitu :

  • Basic Computer Networking

    Basic Computer Networking atau yang biasa dikenal dengan KJD Komputer dan Jaringan Dasar, mempelajari tentang dasar dari sebuah jaringan, seperti mengenal Operasi sistem, Perakitan Komputer hingga K3LH

  • Python Programming

    Python Programming atau biasa dikenal Prodas Pemrograman Dasar, yang satu ini mempelajari tentang basic program seperti, memahami logika pemrograman hingga membuat program sederhana.

  • Basic Graphic Design

    Basic Design Graphics disebut juga DDG Dasar Desain Grafis, mempelajari cara dasar mendesain 2D 3D, mengedit foto hingga membuat ilustrasi Digital.

Core Skills

Beranjak ke tahap selanjutnya yaitu Core Skills, Core Skills ini akan dipelajari di tahun ke 2 dan 3 masa belajar, berbeda dengan basic skills yang mempelajari hal dasar, tahap core skills ini merupakan tahap penting dimana kita akan belajar hal-hal yang akan dibutuhkan di dunia Industri. Yang termasuk diantaranya yaitu :

  • Software as a Service

    Software as a Service, apa aja sih yang akan dipelajari? kita akan mempelajari tentang bagaimana sebuah web dibuat dan bagaimana sebuah web dapat menyimpan data. Mulai dari belajar HTML, CSS, Database hingga Framework Website.

  • Infrastructure as a Service

    Infrastructure as a Service, yang satu ini mempelajari tentang hal-hal yang berkaitan dengan infrastruktur yang akan digunakan dalam membangun sebuah jaringan, seperti pemilihan alat hingga pengoperasian alat hingga dapat membangun sebuah jaringan.

  • Platform as a Service

    Platform as a Service, Nah yang satu ini berkaitan erat dengan yang namanya Server, Mulai dari membuat server virtual hingga membuat service-service yang bisa digunakan client.

  • System Internet of Things

    biasa dikenal dengan SIOT ini mempelajari cara membuat sebuah barang/produk yang dapat mempermudah manusia, seperti membuat jam digital hingga sensor suhu.

  • Network Security

    Network Security, yang satu ini berkaitan dengan keamanan dalam suatu jaringan, kita akan mempelajari bagaimana mengamankan sebuah infrastruktur jaringan dan layanan di dalamnya.

Kepala Program Jurusan SIJA

"SIJA merupakan kompetensi keahlian dengan masa belajar 4 tahun, selama pembelajaran ini kita akan mempelajari 2 kompetensi keahlian : Komputer Jaringan dan Pemrograman, dengan menguasai 2 kompetensi tersebut diharapkan lulusan SIJA bisa dapat langsung bekerja di perusahaan-perusahaan IT Pada kompetensi Keahlian SIJA, ada program andalan tambahan dari kakak-kakak kelas yaitu Nevtik Academy dimana setiap hari sabtu kita dapat mengembangkan skill dan talenta kita di bidang IT, semoga adik-adik berminat untuk bergabung di program keahlian SIJA 4 tahun."
"Mulai tahun 2022, SIJA merupakan bagian dari Program Keahlian Pengembangan Perangkat Lunak dan Gim.
